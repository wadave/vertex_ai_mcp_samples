**1. Introduction**

This guide provides an overview of the ADK Agent and its features.  
It uses a cocktail MCP server to answer your question about cocktail. The MCP server has 5 tools:  
          - 1. search cocktail by name  
          - 2. list all cocktail by first letter   
          - 3. search ingredient by name.   
          - 4. list random cocktails   
          - 5. lookup full cocktail details by id  
**2. Example questions**  
Here are some example questions you can ask the chatbot:  
- 'Please get cocktail margarita id and then full detail of cocktail margarita' 
- 'Please list random cocktails'
